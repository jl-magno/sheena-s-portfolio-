Share Boost

A user-friendly automated website that utilizes your browser's cookies to streamline the process of sharing posts on Facebook

PROCESSING:

How to Get Facebook AppState

1. Download Kiwi Browser

First, download and install Kiwi Browser for Android:

Download Kiwi Browser
Note: Kiwi Browser is needed as it's one of the few Android browsers that can install Chrome extensions.

2. Download FBState Utility

Get the latest version of C3C FBState utility:

Download FBState Utility
Important: Make sure you download the c3c-fbstate-utility-1.5.zip

3. Open Extensions Menu

Install and open Kiwi Browser, then:

  1. Tap the 3 dots (⋮) at the top right
  2. Select "Extensions" from the menu

4. Enable Developer Mode

Find the "Developer mode" switch and slide it to the right to ON

MUST BE ON: The extension won't work if Developer mode is not enabled

5. Install the Extension

Tap the + (from .zip/.crx/.user.js) button, then:

  1. Navigate to where c3c-fbstate-utility-1.5.zip is saved
  2. Find the file with the zipper icon and tap to install
Tip: If the extension is not visible after installation, try restarting Kiwi Browser.

6. Use FBState Utility

After installation:

  1. Login to Facebook.com in kiwi browser (use a dummy account)
  2. Tap the 3 dots (⋮) and enable "Desktop site"
  3. Find and tap "C3C FBState Utility" in the menu
  4. Click "Copy Clipboard" to copy your appstate
Warning: The fbstate is like a key to your Facebook account. Do not share it with anyone!

7. Set Up to Share Boost Site

Now you can use the copied AppState in Share Boost:

  1. Paste your appstate in the form above
  2. Enter your Facebook post link/url
  3. Enter your share limit
  4. Enter your desired sharing speed
  5. Click boost now
